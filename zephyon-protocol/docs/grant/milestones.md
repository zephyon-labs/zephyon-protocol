

# ZephiPay — Grant Milestones

## Milestone 1 — Frontend Refinement & Public MVP Experience

### Objectives
- Finalize the consumer-facing payment interface
- Expand mobile-first transaction usability
- Refine receipt-backed payment confirmation flows
- Improve transaction-state visibility and payment clarity
- Prepare the frontend for broader public-facing demonstration

### Deliverables
- Refined homepage and payment flow UI
- Expanded send / confirm / delivered transaction states
- Improved deterministic receipt presentation
- Public MVP demonstration environment
- Updated screenshots, diagrams, and documentation assets

### Outcome
This milestone finalizes the user-facing payment experience while preserving the simplicity and accessibility required for mainstream blockchain payment usability.

## Milestone 2 — Protocol Hardening, Devnet Expansion & Audit Preparation

### Objectives
- Continue strengthening deterministic receipt infrastructure
- Expand Solana devnet transaction validation
- Improve protocol observability and operational tooling

- Prepare infrastructure and documentation for future third-party security review

### Deliverables
- Expanded devnet payment testing
